## Installation

1. Clone this repository:

```bash
git clone https://github.com/dreambabyyy/Animix-Bot.git
cd Animix-Bot
```

2. Install dependencies:

```bash
npm install 
```

### Setting Up Accounts
```bash
nano data.txt
```
```bash
nano proxy.txt
```
1. Edit `data.txt` file in the root directory
2. Add accessToken (one per line) for each account

```
Query_id/user_id:.......
```


## Usage

Run the bot:

```bash
node main.js

```
